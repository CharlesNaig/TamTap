What is TAMTAP?

TAMTAP is a locally hosted, NFC-based attendance system developed under the capstone project titled "Developing a TAMTAP NFC-Based Attendance System Using Raspberry Pi with Real-Time Dashboard Updates." It is designed specifically for FEU Roosevelt Marikina to provide a secure, efficient, and modern approach to attendance monitoring.

The system is powered by a Raspberry Pi, integrates NFC-based student verification, and supports camera-assisted validation to enhance authenticity. Attendance data is processed and displayed through a real-time dashboard accessible over a local area network (LAN), ensuring fast updates, controlled access, and data privacy without relying on external cloud services.

TAMTAP streamlines attendance tracking while enabling administrators to monitor, review, and backtrack records seamlessly through a secure dashboard interface.

Grade 12 ICT B Capstone Project  |  S.Y. 2025–2026

Key Features

NFC Tap-In

Students tap their NFC card on the reader for a fast, contactless check-in experience under 100ms.

Camera Verification

Face detection via Haar cascade confirms a real person is present — no proxy attendance.

Live Dashboard

Real-time attendance monitoring via Socket.IO on a local network dashboard accessible to teachers and admins.

About This Research

TAMTAP is a Grade 12 ICT B Capstone Project at FEU Roosevelt Marikina for S.Y. 2025–2026. It was developed to streamline student attendance tracking using contactless NFC technology, face detection verification, and real-time monitoring through a local area network.

The system runs entirely on-premises — no cloud services, no internet dependency. Designed for school LAN deployment with data privacy compliance under R.A. 10173.
